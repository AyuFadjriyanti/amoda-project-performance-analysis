# Amoda Performance Analysis Project

## Overview

This project analyzes AMODA’s project performance by evaluating cost efficiency, budget adherence, project duration, and resource utilization.
The goal is to identify inefficiencies, uncover cost leakage, and provide data-driven recommendations to improve operational performance.

## Objectives

- Identify trends in project duration, cost, and resource utilization
- Evaluate budget overruns and cost control consistency
- Measure cost efficiency using Cost Performance Index (CPI)
- Analyze the relationship between resource usage and project cost
- Provide actionable insights for operational improvement

## Dataset

The dataset includes:

- Project category & project name
- Budget vs actual costs
- Resource utilization (hours)
- Project duration & completion timeline
- Cost performance metrics (CPI, budget variance)

## Tools Used

- Data Cleaning & Transformation
- - Spreadsheet / CSV Dataset
- Looker Studio (Dashboard & Visualization)
